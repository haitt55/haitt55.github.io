2. Request a certificate trên ACM

3. Vào load balancer thêm listener asocciate với domain

Vào Ec2: https://ap-northeast-1.console.aws.amazon.com/ec2 > chọn Load Balancers > chọn tab Listeners > HTTPS:443 chọn View/edit certificates sau đó add domain đã verify vào đây
